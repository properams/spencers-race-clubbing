# Asset inbox

Drop ANY downloaded asset files here — HDRIs, PBR texture sets, 3D
models, zipped packs from Quaternius / KayKit / Kenney, individual files
from poly.pizza. Original filenames are fine; I'll inspect, rename, and
move them to the right manifest paths.

**Supported model formats:** `.glb`, `.gltf`, `.obj` (+ optional `.mtl`
+ texture PNGs), `.fbx`. The asset loader auto-detects the extension and
routes to the matching Three.js loader. Just drop whatever you have.

Subfolders are also fine. If you've extracted a Quaternius zip, drop the
whole extracted folder in here and I'll find what I need.

This folder is gitignored — nothing here will be committed.
